# Mercado Libre Electron App
Concept electron app for MercadoLibre

## Run project

#### 🏗️ Build  
```sh
$ npm run build
```

#### 🔥 Run app in development mode
```sh
$ npm run dev
```

#### 💥 Run app
```sh
$ npm install gulp -g
```
